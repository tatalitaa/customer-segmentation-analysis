# Customer Segmentation Analysis (RFM)

## Overview
This project analyzes customer behavior using RFM (Recency, Frequency, Monetary) to identify high-value customers and generate business insights.

## Tools
- Python (Pandas, Matplotlib, Seaborn)

## Key Insights
- Majority of customers are low-frequency buyers
- High-value customers contribute significantly to revenue
- Customer segmentation enables targeted marketing strategies

## Dataset
Online Retail Dataset
